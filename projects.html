Trace

Trace is a Proof of Work (PoW) blockchain-based peer-to-peer transaction platform written in Java. The application allows a group of users to send funds while another group of miners will mine the blocks in exchange for a mining fee. The mined blocks will then be added to the blockchain. I implemented an array list to store all the blocks on the blockchain, a linked dictionary to store the unspent transaction output(UTXOs) objects, SHA256 to calculate the hash value of a block, a Merkle tree to hash the transactions in a block and ECDAS to sign and verify transactions. This project will serve as a basis to my upcoming blockchain-based projects.

FakeBook

FakeBook is a console-based social networking application for university students that allows users to add friends and upload posts. I collaborated with a team of three to develop the app in Java. We created and implemented linked dictionaries to store user objects and post objects, an undirected graph to generate a network between user objects and post objects, a depth-first search stored in a linked bag to display suggested friends and friends posts to the user. We will be implementing this project into a web application in the near future.
